Class Heroes
Link Repo

An application that allow students to participate in class more thoroughly, by using Create, Read, Update, and Delete functionalities(C.R.U.D). Students can message the teacher whenever they need help with school work. Students will provide their emergency contacts, a nickname they prefer to be called, and an update feelings function to the teacher. Teachers can now collect data on how students are feeling, making sure students are attending class, and contacting parents or guardian if necessary

Creative Cocktails
Link Repo

This application targets those who are trying to become bartenders and those who are looking to try new/different types of cocktail drinks. With this app, we provide users information on how to create a specific cocktail drink with Ingredients, Measurements, & Instructions. This app also provides a randomize joke to keep users entertained. As a bartender it is important to know how to create the perfect drink as well as entertain your customers to keep them in a happy mood. Also, if the user is at a bar but wants to try a different drink they can use this app to randomize their options.

Password Generator
Link Repo

This is a password generator. With this application, it provides a secured and randomized password based on Users options. Users will be asked how long they want their password to be, if they want any specific characters in their password such as: numbers, symbols, lowercase letters, uppercase letters, and also if they would like emojis. After completing their options, a randomized password will generate and Users will be able to copy the password directly to their clipboard.
